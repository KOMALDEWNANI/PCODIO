# PCODIO

## Problem Statement
There is no app to aware girls and help them out with PCOD.

### About
An Android Studio application made for SheHacks'21 hackathon.

### Features
* Multi User Login
* PCOD Test
* Exercise Assistance
* Exercise Tracker
* Periods Tracker
* Doctor's Help
* Chatbot

## Tech Stack
* __Development__
    - Android Studio
    - Firebase
* __Design__
  - Canva
